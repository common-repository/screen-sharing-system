Plugin WordPress pour partage d'écran sécurisé et collaboration en temps réel.

== Description ==

**Screen Sharing System** est un plugin WordPress conçu pour faciliter la collaboration en ligne et le support technique. Voici ce qu'il offre :

- **Partage d'Écran Sécurisé**: Se connecter est aussi simple qu'avec un ID unique à 6 chiffres et un mot de passe, offrant une expérience utilisateur inspirée par des leaders du secteur comme TeamViewer.
- **Communication Complète**: Utilisez la webcam, le micro, engagez-vous dans des discussions via chat, et échangez des fichiers de manière illimitée avec une connexion WSS sécurisée.
- **Flexibilité des Paramètres**: Les utilisateurs avertis peuvent ajuster les serveurs ICE et WebSocket pour optimiser la performance selon leurs besoins spécifiques.
- **Design Responsive**: Amélioration du design pour une compatibilité optimale avec tous les appareils, des smartphones aux écrans de bureau.

Ce plugin est idéal pour les consultants, les éducateurs, les équipes de support technique, et quiconque souhaite une solution intégrée pour la collaboration en temps réel sur WordPress.

== Installation ==

1. Téléchargez le plugin depuis WordPress.org.
2. Allez dans 'Plugins' sur votre tableau de bord WordPress.
3. Cliquez sur 'Ajouter Nouveau', puis 'Téléverser Plugin', et sélectionnez le fichier zip du plugin.
4. Installez et activez le plugin.

== Frequently Asked Questions ==

= Comment sécuriser mon partage d'écran ? =
Le plugin utilise un protocole WSS sécurisé et chaque session a un ID unique et un mot de passe.

= Est-ce que ce plugin fonctionne sur toutes les versions de WordPress ? =
Le plugin nécessite au moins WordPress 4.0 et a été testé jusqu'à la version 6.6.2.

= Puis-je utiliser ce plugin pour des sessions de groupe ? =
Actuellement, le plugin est optimisé pour les sessions un-à-un, mais des fonctionnalités supplémentaires sont en développement.
